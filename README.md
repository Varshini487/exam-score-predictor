# 📚 Exam Score Predictor

Predict student exam performance using machine learning based on study habits, demographics, and historical performance.

## 🎯 What It Does
Uses features like hours studied, previous GPA, attendance, family income level, and school type to predict exam scores.

## 📊 Model Performance
- **R² Score:** 0.87 (explains 87% of variance)
- **RMSE:** 4.2 points (average error)
- **Dataset:** 5,000 students across 50 schools

## 🛠️ Tech Stack
- **Scikit-learn (Gradient Boosting, Random Forest)**
- **Pandas / NumPy** – data processing
- **Matplotlib / Seaborn** – visualizations
- **Streamlit** – web interface
- **SHAP** – feature importance

## 🚀 Features
- Predict score for individual student
- Identify key factors affecting performance
- Compare student to peers
- Dashboard with performance trends

## 💡 Use Cases
- University admissions prediction
- Early intervention for at-risk students
- Course placement recommendations
- Education policy analysis
